# 🎬 Cogniflix Frontend

A Netflix-inspired AI streaming UI with modern authentication interface.

## 🚀 Features
- Glassmorphism login UI
- Netflix-style red accent (#E50914)
- Animated background (cinematic feel)
- Framer Motion animations
- OAuth UI (Google / GitHub ready)
- Fully responsive layout

## 🛠 Tech Stack
- React + TypeScript
- Vite
- Framer Motion
- Axios

## 📦 Setup

```bash
npm install
npm run dev
