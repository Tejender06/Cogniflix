# 🎬 Cogniflix Frontend

A premium, Netflix-inspired AI streaming UI built for high performance and smooth user experiences.

## 🚀 Features
- **Premium Cinematic UI:** Dark mode aesthetics with carefully curated colors and glassmorphism elements.
- **Dynamic Interactions:** Hover-reveal metadata on movie cards without clipping, scaling animations, and smooth scrolling.
- **Framer Motion Animations:** Fluid page transitions, animated skeletons for loading states, and dynamic components.
- **AI-Curated Content:** Displays "AI Top Picks For You" directly on the dashboard.
- **Responsive Layout:** Perfectly scales from mobile devices to large widescreen displays.

## 🛠 Tech Stack
- React 19 + TypeScript
- Vite (Fast local development & optimized builds)
- Framer Motion (Animations)
- Lucide React (Icons)
- Custom CSS (for precise styling control, avoiding generic UI frameworks)

## 📦 Setup

```bash
# Install dependencies
npm install

# Start local development server
npm run dev
```
